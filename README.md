PROTOTYPE RADP CONV v1.0
Конвертер аудиоформатов для игры Prototype

Разработано студией FreedomHellVOICE
Руководитель: Никита Шишкин


О ПРОЕКТЕ

Prototype RADP Conv - это профессиональный инструмент для работы с аудиофайлами
игры Prototype. Программа позволяет конвертировать между форматами WAV, RADP и P3D,
а также извлекать аудиоданные из игровых контейнеров.


ВОЗМОЖНОСТИ

Поддерживаемые режимы конвертации:

  [1] WAV → RADP    - Конвертация WAV в сжатый RADP формат
  [2] WAV → P3D     - Конвертация WAV в игровой P3D контейнер
  [3] P3D → WAV     - Извлечение и декодирование аудио из P3D
  [4] P3D → RADP    - Извлечение RADP из P3D контейнера
  [5] RADP → P3D    - Упаковка RADP в P3D контейнер
  [6] RADP → WAV    - Декодирование RADP в WAV
  [7] Извлечь RADP  - Извлечение RADP из P3D (альтернативный режим)

Дополнительные возможности:
  * Пакетная обработка - конвертируйте целые папки с файлами
  * Поддержка IMA ADPCM - качественное кодирование/декодирование аудио
  * Русский и английский интерфейс - переключение на лету
  * Сохранение оригинальных параметров - частота дискретизации, количество каналов

ТЕХНИЧЕСКИЕ ДЕТАЛИ

Форматы:
  WAV  - Стандартный аудиоформат PCM 16-bit (поддерживаются моно и стерео)
  RADP - Проприетарный контейнер с ADPCM сжатием (структура: radp\0RADP)
  P3D  - Игровой контейнер Bohemia Interactive (P3D\xFF заголовок)

Алгоритм сжатия IMA ADPCM:
  * Размер блока: 20 байт (0x14)
  * Сэмплов на блок: 32
  * Таблица шагов: 89 значений (7..32767)
  * Поддержка моно и стерео
  * Динамический шаг квантования

Структура RADP файла:
  [0:5]     - "radp\0"
  [5:9]     - "RADP"
  [9:13]    - Количество каналов (uint32)
  [13:17]   - Частота дискретизации (uint32)
  [17:21]   - Unknown поле (обычно 0x18)
  [21:25]   - Размер ADPCM данных (uint32)
  [25:]     - ADPCM данные (блоки по 20 байт)


ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

Пример 1: Конвертация WAV в P3D для игры

  1. Подготовьте WAV файл (16-bit PCM, моно или стерео)
  2. Запустите скрипт
  3. Выберите режим "WAV → P3D"
  4. Укажите папку с WAV файлами
  5. Нажмите "Пакетная обработка"
  6. Скопируйте полученные P3D файлы в папку ingame/

Пример 2: Извлечение всех аудио из игры

  1. Скопируйте P3D файлы из папки ingame/ в отдельную папку
  2. Запустите скрипт
  3. Выберите режим "Извлечь RADP"
  4. Укажите папку с P3D файлами как входную
  5. Укажите выходную папку для RADP файлов
  6. Затем выберите режим "RADP → WAV"
  7. Конвертируйте RADP в обычные WAV файлы

Пример 3: Одиночная конвертация конкретного файла

  1. Запустите скрипт
  2. Выберите нужный режим
  3. Нажмите "Одиночный файл"
  4. Выберите исходный файл
  5. Укажите имя и место для сохранения результата
  6. Скрипт выполнит конвертацию
